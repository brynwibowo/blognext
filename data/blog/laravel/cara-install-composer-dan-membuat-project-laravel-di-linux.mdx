## Mengaktifkan PHP Extension

Masuk ke terminal **Ctrl**+**Alt**+**T** dan buka file konfigurasi `php.ini` dengan `nano`

```bash
sudo nano /etc/php/php.ini
```

tekan tombol **Ctrl**+**W** lalu ketik `bcmath` dan enter untuk melakukan pencarian

![php.ini](/static/images/php-ini.png)

uncomment atau hilangkan tanda `;` pada `;extension bcmath`, kemudian lakukan hal yang sama pada
`;extension pdo_mysql` dan `;extension zip`, terakhir simpan dengan tekan **Ctrl**+**X** enter

## Install Composer

Composer adalah paket manager PHP yang akan digunakan untuk menginstal laravel dan paket pendukungnya.
Install composer dengan perintah PHP :

```bash
php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');"
```

```bash
sudo php composer-setup.php --install-dir=/bin --filename=composer
```

## Instal atau Membuat Project Laravel

1. **Membuat Project Laravel dengan Composer**

```bash
composer create-project --prefer-dist laravel/laravel laravel-project
```

2. **Membuat Project Laravel dengan Laravel installer**

Untuk proses instal laravel dengan perintah yang singkat bisa menggunakan laravel installer, ikuti langkah berikut :

```bash
composer global require laravel/installer
```

ekspor variabel path

```bash
echo 'export PATH="$PATH:$HOME/.config/composer/vendor/bin"' >> ~/.bashrc
```

```bash
source ~/.bashrc
```

jalankan perintah `laravel`

![php.ini](/static/images/laravel.png)

maka akan muncul pilihan menu pada perintah laravel, untuk membuat project laravel dengan perintah :

```bash
laravel new nama-project-laravel
```

sesuaikan `nama-project-laravel` dengan yang diinginkan.
