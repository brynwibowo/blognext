## Install di Ubuntu dan Mint

```bash
$ sudo apt update
$ sudo apt install apache2
```

## Install di Manjaro dan Arch

```bash
$ sudo pacman -Syu apache
```

## Install di Fedora

```bash
$ dnf install httpd
```
