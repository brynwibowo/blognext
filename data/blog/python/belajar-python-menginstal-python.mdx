## Python

Python merupakan bahasa pemrograman multiguna yang mudah untuk dipelajari baik oleh seorang pemula
maupun yang sudah ahli dalam bahasa pemrograman lain. Penulisan kode python sangat simpel dan _syntax_ pada
python mendekati bahasa manusia sehingga mudah dipahami.
Python pertamakali diperkenalkan pada tahun 1991, python dirancang oleh [Guido Van Rossum](https://id.wikipedia.org/wiki/Guido_van_Rossum)
dan sampai saat ini python terus dikembangkan oleh [Python Software Foundation](https://python.org/psf/).

```python
print("hello world!")
```

hanya dengan satu baris kode python seperti di atas program sudah bisa dijalankan, penulisan kode
tanpa diakhiri tanda semicolon **;** sehingga penulisan kode lebih cepat dan meminimalisir adanya _syntax error_

## Install Python

Python mendukung banyak sistem operasi, untuk sistem operasi berbasis linux hampir semua distro linux sudah terinstal python.
Cek versi Python

```bash
python --version
```

## Percobaan

Buka terminal **Ctrl**+**Alt**+**T**, jalankan python dengan mengetik `python` lalu enter

setelah masuk python shell, tuliskan kode berikut dan tekan enter

```python
print("Halo Python !")
```

atau sebelum masuk python shell buat file python dengan teks editor

```bash
nano belajar-python.py
```

tulis kode program di teks editor lalu simpan dengan menekan tombol **Ctrl**+**X** dan simpan.
Untuk menjalankan program yang dibuat tadi, jalankan perintah di terminal

```bash
python belajar-python.py
```
