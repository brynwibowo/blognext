## Code

1. Buat file python, misal dengan nama file `convertpdf2docx.py`

```python:convertpdf2docx.py
from pdf2docx import Converter
pdf_file = 'samplefile.pdf'
docx_file = 'samplefile.docx'
cv = Converter(pdf_file)
cv.convert(docx_file)
cv.close()
```

2. Tentukan file PDF yang akan dikonversi, misalkan file `samplefile.pdf`
